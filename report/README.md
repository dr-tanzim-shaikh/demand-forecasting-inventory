# Report Folder

This folder contains written report files for the demand forecasting and inventory decision-making project.

## Files

- `results_summary.md` — short summary of forecast accuracy and inventory decision results
- `final_project_report.md` — full project report covering dataset, methods, results, interpretation, limitations, and conclusion

## Report Focus

The reports explain how historical demand data were used to:

- compare forecasting models,
- select the best model,
- calculate safety stock,
- calculate reorder point,
- compare deterministic and uncertainty-aware inventory decisions.
