# Outputs Folder

This folder contains generated tables and plots from the R scripts.

## Folders

- `tables/` — CSV output tables
- `plots/` — generated plots

## Tables Generated

The `tables/` folder contains:

- `data_cleaning_summary.csv`
- `descriptive_summary.csv`
- `monthly_demand_summary.csv`
- `forecast_accuracy.csv`
- `forecast_results.csv`
- `inventory_decision_summary.csv`
- `model_selection_for_inventory.csv`
- `reorder_point_comparison.csv`

## Plots Generated

The `plots/` folder contains:

- daily demand time-series plot
- monthly demand time-series plot
- demand distribution plot
- forecast comparison plot
- ARIMA forecast interval plot
- reorder point comparison plot

## Note

All output files are generated by the R scripts in the `code/` folder.
